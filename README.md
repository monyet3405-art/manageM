# 💸 DuitKu - Aplikasi Manajemen Keuangan Gen Z

Website manajemen keuangan pribadi dengan desain Playful Cards untuk Gen Z, 
dilengkapi fitur gamifikasi seru!

## ✨ Fitur

- **📊 Transaksi** - Catat pemasukan & pengeluaran dengan kategori
- **🎯 Target Tabungan** - Set goal dengan progress bar keren
- **📈 Simulasi** - Hitung pertumbuhan tabungan masa depan
- **🔍 Analisis** - Pie chart & breakdown pengeluaran per kategori
- **🎮 Gamifikasi:**
  - ⚡ XP System - Dapat XP setiap aksi
  - 🏆 Level Up - 9 level dari Pemula hingga Money God
  - 🔥 Streak - Login tiap hari bonus XP
  - 🏅 Badges - 6 achievement badge untuk di-unlock
  - 🎉 Animasi pop-up saat dapat badge/level up

## 🚀 Cara Jalankan

1. Install dependencies:
```bash
pip install flask
```

2. Jalankan server:
```bash
python app.py
```

3. Buka browser ke: `http://localhost:5000`

## 📱 Mobile-Friendly
Didesain untuk kenyamanan di HP dengan:
- Bottom navigation bar
- Touch-friendly buttons
- Responsive layout
- Playful card design

## 🛠️ Teknologi
- **Backend:** Python + Flask
- **Frontend:** Vanilla JS, CSS3
- **Charts:** Chart.js
- **Fonts:** Fredoka + Nunito (Google Fonts)
- **Storage:** JSON file (data.json)
